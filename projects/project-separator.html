Alfa Laval S- and P-flex

This project aimed to develop a software for automatic control of a centrifugal separator. Both control and HMI was developed.

Programming language used was mainly ST, however a few libraries was written in C.

Challenges in this project was that the control system had to be able to detect and adapt to multiple hardware configurations.
